### Professional Self-Assessment

I have improved both my technical skills and my comprehension of what it takes to be a successful software developer during the courses throughout the Computer Science program and the creation of my ePortfolio. My career goals, which are focused on advancing into a development role within my current organization and continuing to develop as a well-rounded engineer, have been shaped by this experience. More significantly, it reiterated that flexibility, problem-solving skills, and continuous learning are necessary for success in this industry.
Working in a collaborative environment is one of the most valuable skills I've picked up. I have learned how to approach development with maintainability and clarity in mind so that others can more easily understand and build on my work through peer discussions, project work, and my professional experience. Additionally, I've become more adept at communicating with stakeholders by clearly and practically outlining technical topics, which is crucial when collaborating with non-technical team members and is something I haven't had the opportunity to do in my professional career thus far.
In terms of technical skills, I have developed a solid foundation in data structures and algorithms, which has strengthened my ability to create effective and valuable solutions. My capstone project, which was originally an Android Studio mobile application developed for my CS-360 Mobile Architecture and Programming class, strengthened my capacity to apply algorithmic thinking to practical issues by implementing a goal weight projection algorithm that evaluates user data to estimate future progress. When porting the app from an Android Studio written in Java, to a React Native application written in JavaScript, I had the opportunity to work with technologies like Node.js and SQL as well. This has given me experience in databases and software engineering on a level that I previously did not have. I also improved my knowledge of data persistence and user-focused design by developing basic features for managing and exporting user data in my Weight Tracker application.
Another crucial factor in my development strategy is security. I've gained an understanding of the significance of secure system design, how to prevent vulnerabilities, and how to protect user data throughout the application. This has improved my comprehension of how security is integrated into the software development process as a whole rather than being handled as a stand-alone issue to be handled towards the end of the development process.
My ePortfolio's artifacts show the variety of skills I've cultivated throughout my degree. The primary artifact is my Weight Tracker mobile application, which demonstrates my abilities to improve an application with both database-driven and algorithmic features. The Travlr Getaways full stack application from my CS-465 Full Stack Development class showcases my experience utilizing the MEAN stack to create a comprehensive web application, whereas my final project report and state machine diagram from my CS-350 Emerging Systems Architectures & Technologies class highlights my capabilities in system design and effective technical communication.
Overall, these artifacts demonstrate my development progress and ability to approach challenges from a variety of perspectives, such as algorithm design, system architecture, and database management. They also illustrate my ability to contribute in a professional setting and continue to create high-quality, effective software solutions.
